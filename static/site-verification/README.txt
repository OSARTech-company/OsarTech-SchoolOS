Place provider verification HTML files in this folder.

Example:
- google1234567890abcdef.html
- bing-site-verification-abc123.html

They will be served from the website root as:
- https://schoolos.osartech.com.ng/google1234567890abcdef.html
- https://schoolos.osartech.com.ng/bing-site-verification-abc123.html

Do not remove the verification file after the provider confirms ownership.
